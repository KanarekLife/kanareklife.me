My Projects

eru

eru is a project created in cooperation with my friend from school Bazik33 and its goal is to create a great platform through which our school would be able to send substitutions quickly. Currently the substitutions are simply uploaded to website and a lot of students misses them. This project includes upload page that dynamically parses file to xml and uploads it into server, from which it automatically sends a notification to registered people via discord and fb messenger. Our sub-goal for this project was to try and grasp as much of clean architecture rules as possible (I'm leading in this task). Project includes using many popular techniques like: Mediator Pattern (achieved via mediatR), CQRS Pattern, Unit testing, Integration testing.

Link to project: Github

WokLearner

WokLearner was another project written for school. Project was written in Angular and .NET Core (as an API). Its goal was to help my fellow students study for end-of-year arts exam which consisted of several painting that you had to name the author and 'age' of creation. Several people registered and used it however due to exam cancellation (due to COVID-19) it did not reached the peak is should have. Application consists of an gallery of paitings and a small quiz that allowed to train for the exam.
